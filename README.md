# IsoLogic

IsoLogic is a Python-based Infection Prevention Isolation & PPE Quick Guide.

## Purpose
This program helps users identify isolation precautions, PPE requirements, and cleaning reminders for common infectious conditions.

## Features
- Menu-driven interface
- Input validation
- Error handling
- Formatted precaution reports
- Repeatable workflow loop

## Python Concepts Used
- Functions
- Dictionaries
- Loops
- Conditional statements
- Error handling

## Conditions Included
- C. difficile
- Influenza
- COVID-19
- Tuberculosis
- MRSA
- RSV
