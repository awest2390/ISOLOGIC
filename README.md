# IsoLogic

## Infection Prevention Isolation & PPE Quick Guide

IsoLogic is a Python-based healthcare decision-support application designed to help users identify appropriate isolation precautions, PPE requirements, and cleaning reminders for infectious conditions.

---

## Project Purpose

The purpose of this project is to simulate a real-world infection prevention workflow using Python programming concepts such as:

- Functions
- Loops
- Dictionaries
- Conditional statements
- Input validation
- Error handling

This project helps standardize isolation precaution recommendations for common infectious diseases.

---

## Features

- Menu-driven interface
- Isolation precaution recommendations
- PPE guidance
- Cleaning and disinfection reminders
- Repeatable workflow loop
- Error handling and input validation

---

## Conditions Included

1. C. difficile
2. Influenza
3. COVID-19
4. Tuberculosis
5. MRSA
6. RSV
